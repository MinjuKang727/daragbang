문제 1) Iteration01.java

문제 설명

변종 3, 6, 9 게임 : 1의 자리에 3,6,9가 있는 경우에만 “짝”을 출력하고, 10의 배수인 경우에는 숫자를 출력하지 않고 다음 숫자로 넘깁니다. 나머지 경우에는 숫자를 그대로 출력해주세요.

출력 예시

n = 30일 때,아래와 같이 출력할 수 있습니다.

1
2
짝
4
5
짝
(7~27까지 생략, 10과 20은 출력하지 않음)
28
짝 // 29까지는 출력하고, 30은 출력하지 않음
public class Iteration01 {

    public static void main(String[] args) {
        // 3~100까지 랜덤으로 연습할 수 지정합니다.
        int n = (int) ((Math.random() * 98) + 3);
        
        // 숫자를 어디까지 연습할 지 출력해봅니다.
        System.out.printf("%d까지 3, 6, 9 게임 연습 시작\n", n);
        
        /**
         * 여기서부터 순서에 따라 구현을 해주세요
         * 1. 반복문 ooo문을 선언합니다. 3, 6, 9 게임은 1부터 n까지 1씩 더하며 반복하면 됩니다.
         * 2. 10의 배수인 경우 다음 숫자로 넘깁니다. 조건문인 oo문과 (i % 10)을 이용하고, oooooooo를 이용해봅니다.
         * 3-1. 1의 자리가 3, 6, 9인 경우를 조건문(oo문)을 통해 찾아봅니다. (i % 10)을 이용하여 나머지를 구하는 것으로 찾아낼 수 있습니다.
         *      -> 3-1의 조건을 만족했다면) "짝"을 출력합니다.
         * 3-2. 2-1의 조건을 만족하지 않은 경우를 위해 조건문(oooo문, 2-1과 세트)을 마저 이용합니다.
         *      -> 3-1 조건을 만족하지 않았다면) 숫자를 그대로 출력합니다.
         */
    }
}

문제 3) Iteration03.java

문제 설명

연속으로 야근한 날짜 출력 : 80% 확률로 야근을 한다고 할 때, 연속으로 야근한 날이 며칠인지 출력해주세요. 단, 합법적으로 연속으로 야근할 수 있는 날은 최대 10일이며, 야근을 전혀 하지 않은 경우는 0일로 계산합니다.

출력 예시

보셨나요? 저의 10일 연속 야근!
public class Iteration03 {

    public static void main(String[] args) {
        // 야근한 날을 계산하기 위해 변수를 선언합니다.
        int overworkDays = 0;
        
        /**
         * 여기서부터 순서에 따라 구현을 해주세요
         * 1. 반복문 ooooo문을 선언합니다. 조건식은 야근 확률인 80%를 구현하기 위해 (Math.random() < 0.8)를 사용할 수 있습니다.
         * 2. 야근할 확률에 당첨되었다면(조건문이 참이라면), 야근한 날을 1일씩 더해줍니다.
         * 3. 야근한 날짜가 10일이 되었는지 oo문을 통해 확인합니다.
         *      -> 10일이 되었다면, 더 이상의 야근은 불법이므로 반복문을 ooooo 키워드를 이용하여 빠져나갑니다.
         */

        // 야근 일 수를 출력해줍니다.
        System.out.printf("보셨나요? 저의 %d일 연속 야근!", overworkDays);
    }
}

문제 1) Array01.java

문제 설명

광역시 입력 받기 : 대한민국에는 서울특별시를 제외하고 총 6개의 광역시가 있습니다. 광역시를 입력받아 배열에 추가해주세요.

입력 예시

광역시 6개를 차례대로 입력해주세요. 엔터로 도시를 구분해주세요.
부산
대전
울산
광주
인천
대구

출력 예시

부산광역시
대전광역시
울산광역시
광주광역시
인천광역시
대구광역시
import java.util.Scanner;

public class Array01 {
    public static void main(String[] args) {

        // 콘솔 입력을 받기 위한 Scanner 객체를 선언합니다.
        Scanner scan = new Scanner(System.in);

        System.out.println("광역시 6개를 차례대로 입력해주세요. 엔터로 도시를 구분해주세요.");

        /**
         * 여기서부터 순서에 따라 구현을 해주세요
         * 1. 광역시를 저장할 크기가 6인 String 배열을 생성, 선언합니다.
         * 2. 광역시를 6개 받아 차례대로 배열에 넣기 위해 배열의 길이만큼 반복하는 ooo문을 작성합니다.
         * 3. scan.nextLine()을 이용하여 콘솔에서 광역시를 받고, 반복문을 통해 배열의 요소에 인덱스로 접근하여 입력한 광역시를 넣어줍니다.
         */
         
        // 향상된 for문으로 배열을 차례로 순회하며 출력합니다.
        for(String city : cities) {
            System.out.printf("%s광역시\n", city);
        }

    }
}

문제 3) Collection03.java

문제 설명

전국의 도시 이름과 소재지 받기 : 이전 Collection01과 02에서는, 단순히 도시 이름만 저장하였기 때문에 이 도시가 어느 도에 위치해 있는지 저장할 수 없었습니다.

따라서 이번에는 도시 이름과 소재지(경기도, 강원도, 충청북도 등)을 key-value 쌍으로 입력하고 저장하여 출력해보도록 하겠습니다.

도시 이름은 유일무이 하지만, 소재지는 겹칠 수 있기 때문에 입력 값에 누락이 없도록 key와 value를 잘 설정해 주어야 합니다.

만약, 도시 이름이 중복되어 들어온 경우에는 소재지가 업데이트 되지 않도록 해주세요.

마찬가지로, exit 입력을 받기 전까지만 도시와 소재지를 저장합니다.

입력 예시

원주 강원도
충주 충청북도
창원 경상남도
수원 경기도
성남 경기도
성남 경상남도

출력 예시
